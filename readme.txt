  Product Enquiry allows prospective customers to “Make an Enquiry” about a product, right from within the product page.

== Description ==

The  Product Enquiry for WooCommerce plugin adds an enquiry button to every WooCommerce Product Page. Using this button a potential customer, can send you an enquiry or request a quote for a product he is unsure of purchasing. By addressing his enquiries or quote requests, you can convert an unsure customer, into a buyer!

= Features =

**Add an Enquiry or Request a Quote Button**: Allows customers or visitors to make enquiry about a product by asking question about a product directly from the product listing page.


== Installation ==

Installation via WordPress Dashboard

1. Go to your WordPress Dashboard and click on “Plugins”

2. Click on “Add New” and search for “ Woocommerce Product Inquiry ”.

3. Click on Install and activate the plugin.


Installation via FTP

1.	Download the plugin files and extract to the '/wp-content/plugins/' directory. 

2.	Go to your WordPress Dashboard, and activate the plugin from the Plugins page.

3.	Plugin should work out of the box, no settings necessary.
